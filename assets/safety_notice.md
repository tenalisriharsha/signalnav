# Safety Notice

SignalNav is designed for hands-free use only.

## Key Safety Features
- UI locks automatically when speed exceeds 5 mph
- All guidance is audio-based in driver mode
- Speed recommendations never exceed posted limits
- No countdown timers that encourage rushing

## Your Responsibility
- Always obey traffic signals and laws
- Do not use hand-held devices while driving
- Use voice commands or ask a passenger to interact with the app
- Pull over safely if you need to use the map or reporting features

## Illinois Law
Illinois prohibits the use of hand-held electronic devices while driving. This app is designed to comply with hands-free requirements.
